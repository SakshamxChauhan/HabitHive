# HabitFlow – Daily Habit Tracker Web Application

## 📌 Overview

HabitFlow is a simple and interactive web application designed to help users track and manage their daily habits. It demonstrates core web development concepts including JavaScript, API integration, and responsive UI design.

The application allows users to view, search, filter, and sort habits, as well as mark them as completed or incomplete, making it both functional and user-friendly.

---

## 🚀 Features

* 🔄 Fetch habit/task data using a public API
* 🔍 Search habits by name
* 🎯 Filter habits (Completed / Incomplete)
* ↕️ Sort habits alphabetically (A–Z)
* ✅ Mark habits as completed or pending
* 📊 Progress tracking (completed vs total tasks)
* 💡 Clean and responsive user interface

---

## 🛠️ Technologies Used

## ⚙️ How It Works

1. The application fetches habit/task data from a public API.
2. The data is stored in an array.
3. JavaScript array methods are used to:

   * Display tasks (`map()`)
   * Filter tasks (`filter()`)
   * Sort tasks (`sort()`)
4. Users interact with the UI to manage their habits dynamically.
